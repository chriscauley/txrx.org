jQuery Thumb Scroller

help guide

Installation

To install the thumb scroller, first include the style sheet inside the <head> tag: 

<link rel="stylesheet" type="text/css" href="css/thumb-scroller.css" />

Next, include the jQuery's library file, the jQuery UI file, and the thumb scroller's script file inside the <head> tag: 

<script type="text/javascript" src="js/jquery-1.10.2.min.js"></script>
<script type="text/javascript" src="js/jquery-ui-1.10.3.custom.min.js"></script>
<script type="text/javascript" src="js/jquery.thumb-scroller.js"></script>

This completes the installation process, next you'll have to specify the HTML markup.


HTML Markup

To insert a thumb scroller for your page, specify the thumb scroller's markup inside the <body> tag. The following is an example of a basic markup: 

<div id="scroller1" class="thumb-scroller">
    <ul class="ts-list">
        <li>
            <img src="images/photo1.jpg"/>
        </li>                    
        <li> 
            <img src="images/photo2.jpg"/>
        </li>
        <li> 
            <img src="images/photo3.jpg"/>
        </li>
    </ul>
</div>

From the example, the <div> tag on line 1 represents the thumb scroller's container. The <div> tag should have the required class thumb-scroller but can have any id value. For this example, we use an id of scroller1 which will be referenced in the activation function call.

Also of importance is the <ul> tag with class ts-list. This <ul> tag will hold all the <li> tags, with each <li> tag representing an individual slide.


Images

There are two ways to add an image to the thumb scroller. One is using a regular <img> tag and the other is specifying an url to the image.

For the first method, just insert an <img> tag inside the <li> tag: 

<li>
    <img src="images/photo1.jpg" />
</li>

This is the simpler of the two methods but the image is downloaded immediately versus when needed.


For the second method, to add an image using a link, insert a data-src attribute for the <li> tag: 

<li data-src="images/photo1.jpg">
    <img src="images/placeholder.jpg" />
</li>

This method will reduce the initial download time using lazy loading. The image is downloaded as needed instead of all at once.

From the example, once photo1.jpg is finished loading, it will replace placeholder.jpg. It is recommended you use a small place holder image, preferably a transparent 1 x 1 pixel image.


Other Contents

Aside from images, you can load other types of content for a slide. A slide can support content coming from flash, iframe, ajax, and inline content.

In order to load other types of content, assign the content's url to the data-src attribute for the <li> tag. You can also specify an optional data-content-type attribute to tell what type of content it is loading. If you do not specify a data-content-type attribute, the scroller will try its best to determine what type it is.

Flash

To load flash content, specify the swf file for the data-src attribute. 

<li data-src="test.swf" data-content-type="flash">
    <img src="images/placeholder.jpg" />
</li>

iframe

To load content for an iframe, specify an url for the data-src attribute. 

<li data-src="http://www.codecanyon.net" data-content-type="iframe">
    <img src="images/placeholder.jpg" />
</li>

AJAX

To load content for an ajax call, specify an url with your variables for the data-src attribute and set the data-content-type attribute to ajax. 

<li data-src="test.php?var1=hello&var2=world" data-content-type="ajax" data-method="POST">
    <img src="images/placeholder.jpg" />
</li>

You can also specify either a GET or POST type using the data-method attribute.

Inline Content

To load inline content for the slide, specify an id value for the data-src attribute. 

<li data-src="#mydiv" data-content-type="inline">
    <img src="images/placeholder.jpg" />
</li>

The id should reference the id of the element you want the html content to come from.


Links

To add a link for a slide, insert a <a> tag around your image. 

<li>
    <a href="http://www.codecanyon.net"><img src="images/photo1.jpg" /></a>
</li>

Captions

To add a caption for a slide, insert a tag like div within the <li> tag. 

<li>
    <img src="images/photo1.jpg" />
    <div>
    	Lorem ipsum dolor sit amet.
    </div>
</li>

You can use virtually any tags to represent a caption. Some supported tag types are <div>, <p>, <span>, and <h1> thru <h6>.

Also, you can embed html content within the caption tag like in the following example: 

<li>
    <img src="images/photo1.jpg" />
    <div>
    	<a href="http://www.codecanyon.net">Code Canyon</a>
    	<em>Lorem ipsum dolor sit amet.</em>
    </div>
</li>

Caption Configuration

You can configure each caption individually. To customize these options, set the following data attributes for the <li> tag: 

<li data-caption-align="top" data-caption-effect="fade" data-caption-button="true">
    <img src="images/photo1.jpg" />
    <div>
    	lorem ipsum
    </div>
</li>

The available options are:

  • data-caption-align - the caption's alignment. (top or bottom)
  • data-caption-effect - the caption's transition effect. (none, normal, fade, or slide)
  • data-caption-button - specify if the caption's toggle button is displayed. (true or false)

Image Position

You can configure the positioning of each slide image by setting the data-image-position attribute of the <li> tag: 

<li data-image-position="center">
    <img src="images/photo1.jpg" />
</li>

The available values for data-image-position are:

  • center - center the image within the slide container.
  • fill - fill the slide container with the image. (cropping might occur)
  • fit - fit the whole image within the slide container.
  • stretch - stretch the image to fill the slide container.
  • none - show the image as is.


Activation & Configuration

Once the HTML markup is specified, the final step is to activate it to get it running. To activate the thumb scroller, simply use the following script in your page: 

<script type="text/javascript">
$(window).load(
   function() {
      $("#scroller1").thumbScroller();
   }
);
</script>

The code snippet activates the thumb scroller by using the id of the thumb scroller's <div> container tag and applying the activation function on it. Also, make sure the activation function call is within the $(window).load() function.


The thumb scroller has many configurable options. In order to configure the thumb scroller, pass in a list of the options to the activation function call. The following example activates the thumb scroller, but this time with the option slideWidth and slideHeight set. 

<script type="text/javascript">
$(window).load(
    function() {
        $("#scroller1").thumbScroller({
            slideWidth:400,
            slideHeight:300
        });
    }
);
</script>

Please note that any options not specified for the function call will take on its default value. For more information on all available options, please refer to the Options tables.

After activation and configuration, your thumb scroller is ready to run.


Options

Size

Name Description Default Value Possible Values
numDisplay The number of slides visible 4 Any positive integer
slideWidth The slide's width 300 Any positive integer
slideHeight The slide's height 200 Any positive integer
slideBorder The slide's border size 0 Any non-negative integer
slideMargin The slide's margin size 0 Any non-negative integer
padding The scroller's surrounding padding 10 Any non-negative integer
captionHeight The caption's height auto Any positive integer or auto

Slideshow

Name Description Default Value Possible Values
startIndex The slide index to start on. 0 Any non-negative integer
autoPlay Set true for slide show to play automatically on startup. false true, false
delay The slide show's delay in milliseconds. 5000 Any positive integer
speed The scrolling transition speed in milliseconds. 800 Any positive integer
easing The transition animation easing. swing See jQuery Easing for information on available easing.
pauseOnHover Set true to pause slide show on mouseover and play on mouseout. false true, false
pauseOnInteraction Set true to pause slide show on control's interaction. false true, false
playOnce Set true for slide show to play through only once, stopping at the last slide. false true, false
continuous Set true for navigation go to beginning again once end is reached. true true, false
moveByOne Set true for scrolling to move by one slide at a time. false true, false

Navigation Control

Name Description Default Value Possible Values
control The type of control. index none, index, scrollbar
navButtons The type of navigation buttons. normal none, small, normal, mouseover
playButton Set true to display the play button. true true, false
pageInfo Set true to display the current number info. true true, false
title The header title. Any string value

Captions

Name Description Default Value Possible Values
captionAlign The alignment of the slide captions. bottom top, bottom
captionPosition The position of the slide captions. inside inside, outside
captionEffect The caption transition effect. fade none, normal, fade, slide
captionButton Set true to display caption toggle buttons. false true, false

Miscellaneous

Name Description Default Value Possible Values
orientation The orientation of the scroller. horizontal horizontal, vertical
responsive Set true for scroller to be responsive. Set false for fixed size. false true, false
collapsible Set true for scroller to be collapsible and expandable when window resizes. Only available for horizontal orientation. Note: if true, responsive setting is overrided. false true, false
minDisplay The minimum number of slides displayed. Note: Only applicable if collapsible is set to true. 1
imagePosition The position of the slide images. fill none, center, fill, fit, stretch
shuffle Set true to shuffle slides in random order. false true, false
keyboard Set true to enable keyboard navigation. false true, false
mousewheel Set true to enable mousewheel scroll navigation. false true, false

Callbacks

Name Description Default Value Possible Values

onPrevious Called when advancing to previous slides. function() {} Any javascript function
onNext Called when advancing to next slides. function() {} Any javascript function
onBegin Called when the first slide is reached. function() {} Any javascript function
onEnd Called when the last slide is reached. function() {} Any javascript function
